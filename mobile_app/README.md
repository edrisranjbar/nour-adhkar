# Nour Adhkar - Mobile App

Islamic prayers and remembrances app - Android version.

## Platform

This Flutter project is configured for **Android only**.

## Getting Started

### Prerequisites

- Flutter SDK
- Android Studio or VS Code with Flutter extensions
- Android SDK

### Building for Android

```bash
flutter pub get
flutter build apk
# or for app bundle
flutter build appbundle
```

### Running the app

```bash
flutter run
```

## Features

- RTL (Right-to-Left) support for Persian/Farsi
- Localized app name (Persian/English based on device language)
- Beautiful UI matching the Vue.js web app
- Component-based architecture
